# 📱 Тестування на iPhone 12

## 🔧 Налаштування для iPhone 12

### Екран iPhone 12:
- **Розмір:** 390 × 844 пікселів
- **Співвідношення:** 19.5:9
- **Notch:** є (виріз для камери)
- **Safe Area:** є зони відступів

## 🌐 Як відкрити додаток на iPhone

### Спосіб 1: Через локальну мережу
1. Переконайтеся, що iPhone та ноутбук в одній WiFi мережі
2. Відкрийте на iPhone: **http://192.168.192.20:3001**
3. Додайте в закладки для швидкого доступу

### Спосіб 2: Встановлення як PWA
1. Відкрийте додаток в Safari
2. Натисніть **"Поділитися"** (квадрат зі стрілкою вгору)
3. Прокрутіть вниз і натисніть **"На домашній екран"**
4. Натисніть **"Додати"**
5. Іконка з'явиться на домашньому екрані

## 🚨 Тестування функцій

### 1. **SOS кнопка**
- Перевірте розмір та доступність
- Тестування анімації pulse
- Перевірка натискання

### 2. **Emergency Mode**
- Дозвіл на камеру (обов'язково!)
- Перевірка відеозапису
- Геолокація (дозвіл обов'язково!)
- Юридичні питання (audio)

### 3. **Інциденти**
- Прокрутка списку
- Натискання на картки
- Модальні вікна

### 4. **Навігація**
- Переключення між сторінками
- Мобільне меню
- Кнопка "Назад"

## ⚠️ Важливі налаштування iPhone

### Дозволи які потрібні:
1. **Камера** - для відеозапису
2. **Мікрофон** - для аудіо
3. **Геолокація** - для координат
4. **Хмарне сховище** - для синхронізації

### Як налаштувати:
1. **Налаштування** → **Конфіденційність та безпека**
2. **Камера** → знайдіть додаток → дозволіть
3. **Мікрофон** → знайдіть додаток → дозволіть  
4. **Служби локації** → знайдіть додаток → "Під час використання"

## 🔍 Проверка проблем

### Якщо щось не працює:
1. **Оновіть сторінку** - pull to refresh
2. **Перевірте дозволи** в налаштуваннях
3. **Очистіть кеш Safari**
4. **Перезапустіть Safari**

### Поширені проблеми:
- **Камера не працює** → перевірте дозвіл
- **Геолокація не визначається** → увімкніть GPS
- **Сторінка не завантажується** → перевірте WiFi з'єднання
- **Кнопки не натискаються** → оновіть сторінку

## 📊 Тестування продуктивності

### Що перевірити:
- **Швидкість завантаження** - має бути < 3 секунди
- **Плавність анімацій** - без затримок
- **Прокрутка** - плавна без підгальмовувань
- **Пам'ять** - додаток не має "вилітати"

## 🎯 Режими екрана

### Portrait (390 × 844):
- SOS кнопка велика
- Одноколонковий layout
- Великі відступи

### Landscape (844 × 390):
- Двоколоночний layout
- Компактні кнопки
- Оптимізована висота

## 📱 Поради для тестування

1. **Тестуйте в обох режимах** (portrait/landscape)
2. **Перевірте всі дозволи** перед тестуванням
3. **Використовуйте реальні умови** (не тільки WiFi)
4. **Тестуйте з різною яскравістю екрана**
5. **Перевірте роботу з іншими додатками** (multitasking)

## 🚀 PWA функції

### Після встановлення на домашній екран:
- Запускається як нативний додаток
- Повноекранний режим без Safari
- Працює офлайн (частково)
- Швидкий доступ до SOS
