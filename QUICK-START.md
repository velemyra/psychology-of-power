# 🚀 Швидкий старт - ПСИХОЛОГІЯ СИЛИ

## Ваша конфігурація:
- 🖥️ **Ноутбук:** Windows з Chrome/Edge
- 📱 **Телефон:** iPhone 12 з Safari
- 🌐 **Мережа:** Локальна Wi-Fi

## ⚡ Запуск за 2 хвилини

### 1. На Windows ноутбуці:
```bash
# Відкрийте CMD/PowerShell
cd C:\Users\user\CascadeProjects\psychology-of-power
npm install
npm run dev:local
```

### 2. Отримайте IP адресу:
```bash
# В тому ж терміналі:
ipconfig
# Знайдіть "IPv4 Address" (наприклад: 192.168.1.100)
```

### 3. На iPhone 12:
```
1. Відкрийте Safari
2. Перейдіть на http://192.168.1.100:3000
3. Натисніть "Поділитися" → "На домашній екран"
4. Натисніть "Додати"
```

### 4. На Windows:
```
1. Відкрийте Chrome або Edge
2. Перейдіть на http://localhost:3000
3. Натисніть іконку встановлення ⬇
4. Натисніть "Встановити додаток"
```

## ✅ Що тестувати в першу чергу

### 🚨 Екстрений режим
- **iPhone:** Натисніть SOS → перевірте камеру та геолокацію
- **Windows:** Натисніть SOS → перевірте якість відео

### 📹 Інциденти
- **Створіть** інцидент на одному пристрої
- **Перевірте** доступність на іншому

### 🔄 Синхронізація
- **iPhone:** Створіть інцидент офлайн (Airplane Mode)
- **Windows:** Перевірте появу після підключення

### iPhone 12:
- ✅ **A14 Bionic** процесор - швидка обробка відео
- ✅ **Super Retina** дисплей - ідеальна якість

- ✅ **GPS** - висока точність геолокації

- ✅ **File System** - легке збереження даних
- ✅ **System Integration** - сповіщення, ярлики
- ✅ **Keyboard Shortcuts** - Ctrl+S, Ctrl+E, F11

### Синхронізація:
- ✅ **IndexedDB** - офлайн зберігання

- ✅ **Cross-platform** - дані доступні скрізь


```
Windows Settings → Privacy → Microphone → Allow apps
Windows Settings → Privacy → Location → Allow apps
```
