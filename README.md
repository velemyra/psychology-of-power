# ПСИХОЛОГІЯ СИЛИ - PWA Додаток

## Опис

**PWA додаток для захисту прав водіїв під час спілкування з поліцією** з функціями фіксації порушень, юридичною підтримкою та інструментами для документування інцидентів.

## 🚀 Основні функції

### 🚨 Екстрений режим
- **SOS кнопка** - миттєве активування при зупинці поліцією
- **Відеозапис** - одночасна зйомка з фронтальної та основної камери
- **Геолокація** - автоматичне фіксування координат та часу
- **Юридичні питання** - аудіо-кнопки з професійними питаннями до поліції

### 📁 Інциденти
- Хронологічний список всіх записаних інцидентів
- Детальна інформація: відео, локація, час, нотатки
- Пошук та фільтрація
- Експорт та поширення

### 📝 Генератор скарг
- Шаблони скарг до поліції, ДБР, СБУ
- Автоматичне заповнення даних інциденту
- Посилання на закони та статті
- Експорт у PDF/Word

### 📂 Докази
- Об'єднання відеозаписів та скарг
- Можливість відправки на пошту, месенджери, соцмережі
- Шифрування даних

### 💰 Калькулятор штрафів
- Повна база штрафів ПДР
- Сума, стаття, відповідальність
- Пошук за категорією

### 🚦 Дорожні знаки
- Сканування через камеру
- Детальна інформація про знаки
- Штрафи за порушення

### ⚖️ Юридична допомога
- Пояснення статей ПДР та законів
- Практичні поради
- Шаблони відповідей

## 💎 Підписки

### Free (Безкоштовно)
- До 5 інцидентів на місяць
- Базові шаблони скарг
- Калькулятор штрафів
- Дорожні знаки (офлайн)

### PRO (99 грн/місяць)
- Необмежена кількість інцидентів
- Хмарне зберігання на 30 днів
- 50+ шаблонів скарг
- Пріоритетна підтримка
- Автоматична відправка скарг

### Premium (199 грн/місяць)
- Все з PRO +
- Консультації юристів (2 на місяць)
- Персоналізовані шаблони
- Аналіз інцидентів юристом
- Знижка 30% на представництво в суді

## 🛠 Технології

- **Frontend**: React 18 + Vite
- **PWA**: Service Worker, Manifest, Offline підтримка
- **База даних**: IndexedDB (офлайн) + Firebase (хмара)
- **Стилі**: CSS-in-JS, адаптивний дизайн
- **Іконки**: Lucide React
- **Хостинг**: Vercel/Firebase Hosting

## 📱 Встановлення

### Онлайн
1. Відкрийте додаток у браузері
2. Натисніть "Встановити додаток"
3. Підтвердіть встановлення

### Офлайн (розробка)
```bash
# Клонуйте репозиторій
git clone https://github.com/your-username/psychology-of-power.git
cd psychology-of-power

# Встановіть залежності
npm install

# Запустіть розробку
npm run dev

# Збірка для продакшену
npm run build
```

## 🔧 Налаштування

### Environment Variables
Створіть файл `.env.local`:
```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

### Firebase
1. Створіть проєкт у [Firebase Console](https://console.firebase.google.com/)
2. Увімкніть Firestore Database
3. Налаштуйте Authentication
4. Активуйте Cloud Functions (опційно)

## 📲 PWA функції

### Offline підтримка
- Базові функції працюють без інтернету
- Автоматичне збереження даних
- Синхронізація при підключенні

### Push сповіщення
- Оновлення законодавства
- Нові шаблони скарг
- Нагадування про підписку

### Встановлення
- Підтримка на Android/iOS
- Робота в standalone режимі
- Повноекранний режим

## 🌐 Деплой

### Vercel
```bash
# Встановіть Vercel CLI
npm i -g vercel

# Розгорніть проєкт
vercel --prod
```

### Firebase Hosting
```bash
# Встановіть Firebase CLI
npm i -g firebase-tools

# Ініціалізуйте проєкт
firebase init hosting

# Розгорніть
firebase deploy --only hosting
```
