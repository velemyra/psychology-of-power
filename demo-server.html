📱 Тестування камери

Це демо сторінка для перевірки роботи камери


📊 Статус:

Натисніть "Запустити камеру" для тестування

❌ Помилка:

🔧 Інструкції:

1. Натисніть "Запустити камеру"

2. Надайте дозвіл на камеру

3. Перевірте чи працює відео

4. Якщо працює - основний додаток також буде працювати

📱 Для iPhone:

• Потрібен HTTPS протокол

• Надайте дозвіл в налаштуваннях iOS

• Перевірте чи камера не використовується іншими додатками
